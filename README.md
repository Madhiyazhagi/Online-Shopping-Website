# MoonBeauty Shop - Online Shopping Website

MoonBeauty Shop is a fully responsive, ultra-professional online shopping website built with **HTML, CSS, and JavaScript**. This project showcases a complete e-commerce experience including browsing products, adding items to the cart, and a responsive user interface. It is designed to impress HRs and demonstrate front-end development skills.

## Features

- **Responsive Design:** Works perfectly on desktop, tablet, and mobile devices.
- **Hero Section:** Stunning homepage banner with promotional content.
- **Featured Products:** Product grid with images, prices, and Add to Cart buttons.
- **Cart Functionality:** Dynamic cart updates with quantity and total price calculation.
- **Login/Register:** Interactive user forms for authentication.
- **Newsletter Subscription:** Collect user emails for updates and promotions.
- **Professional Styling:** Modern fonts, hover effects, buttons, and animations.
- **Favicon & Branding:** Brand identity integrated into the browser tab.
- **Clean Folder Structure:** Organized assets (images, CSS, JS) for easy deployment.

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Google Fonts (Poppins)**
- **Font Awesome Icons**
